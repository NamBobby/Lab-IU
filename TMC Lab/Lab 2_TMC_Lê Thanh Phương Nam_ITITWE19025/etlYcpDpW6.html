 Given

f (x) = −2x6 − 1.6x4 + 12x + 1

Use bisection to determine the maximum of this function. Employ

initial guesses of xl = 0 and xu = 1, and perform iterations until

the approximate relative error falls below 5%.

The maximum will lie between 0.875 and 0.90625

Step-by-step explanation

For a relative error of 5% , we need 5 / 100 * (1 - 0) *100 = 5 iterations


First Iteration :


Step 1 : Calculate midpoint = Xm = (Xu + Xl) / 2 = (1 + 0) / 2 = 0.5

Now take 2 values just below and above this midpoint : 0.4 and 0.6


f(x) at 0.4 = 2(0.4)^6 − 1.6x(0.4)^4 + 12(0.4) + 1 = 5.751

f(x) at 0.6 = 2(0.6)^6 − 1.6x(0.6)^4 + 12(0.6) + 1 = 7.899


Since f(0.6) > f(0.4) function is increasing and hence new

Xl = Xm = 0.5

Xu = 1


Second Iteration :


Step 1 : Calculate midpoint = Xm = (Xu + Xl) / 2 = (1 + 0.5) / 2 = 0.75

Now take 2 values just below and above this midpoint : 0.8 and 0.7


f(x) at 0.8 = 2(0.8)^6 − 1.6x(0.8)^4 + 12(0.8) + 1 = 9.42

f(x) at 0.7 = 2(0.7)^6 − 1.6x(0.7)^4 + 12(0.7) + 1 = 8.781


Since f(0.8) > f(0.7) function is increasing and hence new

Xl = Xm = 0.75

Xu = 1


Third Iteration :


Step 1 : Calculate midpoint = Xm = (Xu + Xl) / 2 = (1 + 0.75) / 2 = 0.875

Now take 2 values just below and above this midpoint : 0.87 and 0.88


f(x) at 0.88 = 2(0.88)^6 − 1.6x(0.88)^4 + 12(0.88) + 1 = 9.672

f(x) at 0.87 = 2(0.87)^6 − 1.6x(0.87)^4 + 12(0.87) + 1 = 9.656


Since f(0.88) > f(0.87) function is increasing and hence new

Xl = Xm = 0.875

Xu = 1


Fourth Iteration :


Step 1 : Calculate midpoint = Xm = (Xu + Xl) / 2 = (1 + 0.875) / 2 = 0.9375

Now take 2 values just below and above this midpoint : 0.94 and 0.93


f(x) at 0.94 = 2(0.94)^6 − 1.6x(0.94)^4 + 12(0.94) + 1 = 9.651

f(x) at 0.93 = 2(0.93)^6 − 1.6x(0.93)^4 + 12(0.93) + 1 = 9.669


Since f(0.93) > f(0.94) function is decreasing and hence new

Xl = 0.875

Xu = Xm = 0.9375


Fifth Iteration :


Step 1 : Calculate midpoint = Xm = (Xu + Xl) / 2 = (0.9375 + 0.875) / 2 = 0.90625

Now take 2 values just below and above this midpoint : 0.907 and 0.905


f(x) at 0.905 = -2(0.905)^6 − 1.6x(0.905)^4 + 12(0.905) + 1 = 9.688

f(x) at 0.907 = 2(0.907)^6 − 1.6x(0.907)^4 + 12(0.907) + 1 = 9.689


Since f(0.907) > f(0.905) function is decreasing and hence new

Xl = 0.875

Xu = Xm = 0.90625


Therefore the maximum will lie between 0.875 and 0.90625
